# 🎯 Jogo da Adivinhação em Python

Um pequeno projeto em **Python** onde o jogador precisa adivinhar um número secreto gerado aleatoriamente entre **1 e 100**.

O programa informa se o número digitado é **muito alto** ou **muito baixo**, até o jogador acertar.
Ao final, o sistema mostra **quantas tentativas foram necessárias**.

---

## 🚀 Tecnologias utilizadas

* Python 3
* Biblioteca padrão `random`

---

## 🧠 Como funciona

1. O programa gera um número aleatório entre **1 e 100**
2. O usuário digita um número
3. O sistema informa se o número é:

   * 🔼 **Muito alto**
   * 🔽 **Muito baixo**
4. O jogo continua até o usuário acertar
5. No final é exibido o **número total de tentativas**

---

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/mbrv-dev/jogo-adivinhacao-python.git
```

2. Entre na pasta do projeto:

```bash
cd jogo-adivinhacao-python
```

3. Execute o programa:

```bash
python jogo_adivinhacao.py
```
